=======   MixMods.com.br   ======= - Melhor visualizável no bloco de notas com fontes mono-espaçadas, como "Lucida Console".


------- Instruções:

Extraia a pasta "cleo" para a pasta do seu GTA SA. (nos testes NÃO funcionou ao instalar pelo ModLoader!)
 Caso SAMP, use SAMP FXT Fix: http://www.mixmods.com.br/2016/08/samp-fxt-fix-corrigir-mods-fxt-no-samp.html

 NECESSÁRIO NewOpcodes.cleo: https://drive.google.com/file/d/0B1sAJQAUfOOYckxKaVZ4NjZJa1U/view



- Problemas e soluções
 Altamente recomendado o uso da CLEO 4.3 ou mais nova, para evitar bugs como o ícone da arma bugando: http://www.mixmods.com.br/2014/02/Livraria-CLEO.html
 Não use junto com outros mods de hud (exceto casos menores como detalhes no hud, velocímetros etc).
 Causa crash caso você utilize algum ajuste de limite de arquivos .dff (pode acontecer em certos casos como conversões totais).


- Configure o mod
 Sinta-se livre para editar os arquivos da pasta "cleo\aceHUD" do mod para personalizar o seu hud. Digite "ACELOAD" para recarregar o .ini dentro do jogo. Há também o comando "T" para mostrar informações ocultas.
 Dentro do arquivo "Readme (ORIGINAL).txt" você tem mais informações sobre o mod.


- Pré-configurações
 Dentro da pasta "presets" contém as pastas com todos os arquivos necessários para cada estilo de hud.
 Para instalar, simplesmente substitua a pasta "cleo" pela nova. Alguns presets também vem com pasta "data" e "models", neste caso é recomendado instalar pelo ModLoader para assim não substituir seus arquivos e ser fácil desinstalar.


- Armas adicionadas sem substituir
 Caso você queira adaptar armas adicionadas sem substituir para este hud:

  Caso você use hud configurado como "UseInternalIcons = 0" (hudSetting.ini) (para usar ícone por imagem .png)
   1- Abra o "hudWeapons.ini" e procure por "last_weapon_id = 46". Troque o "46" pelo maior ID de arma que você tenha adicionado (provavelmente você usará "70", ou "71" etc dependente a quantidade de armas que você tenha adicionado)
   2- Coloque uma imagem .png do ícone da arma na pasta "cleo\aceHUD\weapon" com algum nome (por exemplo "revolver").
   3- No mesmo arquivo .ini, logo abaixo de "46 = gun_para" coloque o ID da sua arma seguido do nome do ícone, por exemplo "70 = revolver".

  Caso você use hud configurado como "UseInternalIcons = 1" (hudSetting.ini) (para usar o ícone dentro do .txd da arma)
   1- Abra o "hudWeapons.ini" e vá até a última linha do arquivo.
   2- Adicione uma nova linha dizendo "<id do modelo> = <nome do .txd>", por exemplo "12400 = revolver".

  Caso você não saiba qual ou use diferentes configurações de hud (por padrão vem "UseInternalIcons = 0"), simplesmente adapte ambos os casos acima, assim você não precisa mais se preocupar com isso.
